# 🕸️ Web News Scraper
Ένα εργαλείο συλλογής δεδομένων (Data Collection) που εξάγει τίτλους ειδήσεων από το διαδίκτυο.

## 🛠️ Τεχνολογίες
- **Python 3**
- **BeautifulSoup4**: Για την ανάλυση του HTML.
- **Requests**: Για τη σύνδεση με την ιστοσελίδα.

## 🧠 Τι έμαθα
- Πώς να στέλνω HTTP requests.
- Πώς να αναζητώ συγκεκριμένα HTML tags (όπως `<h3>` ή `<a>`).
- Τη σημασία του αρχείου `robots.txt` για το νόμιμο scraping.
