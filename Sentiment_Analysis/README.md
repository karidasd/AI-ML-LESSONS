## 📂 Projects

### 1. Sentiment Analysis Tool 🎭
A web application that uses Natural Language Processing (NLP) to determine the emotional tone of a given text.

* **Technology:** Python, Streamlit, TextBlob.
* **How it works:** It calculates a **polarity score** (from -1 to 1) to classify text as Positive, Negative, or Neutral.
* **Location:** `/Sentiment_Analysis`
