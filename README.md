🤖 AI-ML Lessons & Projects
Αυτό το αποθετήριο αποτελεί έναν κεντρικό κόμβο για την εκμάθηση και την πρακτική εφαρμογή αλγορίθμων Τεχνητής Νοημοσύνης (AI) και Μηχανικής Μάθησης (ML). Περιλαμβάνει κώδικα, notebooks και πειραματικές εφαρμογές που καλύπτουν ένα ευρύ φάσμα σύγχρονων τεχνολογιών.

🛠️ Περιβάλλον Εργασίας
Για την ομαλή εκτέλεση των ασκήσεων και των μοντέλων, προτείνεται η χρήση Python 3.10+.

Εγκατάσταση Εξαρτήσεων: Πριν ξεκινήσετε, εγκαταστήστε τις απαραίτητες βιβλιοθήκες χρησιμοποιώντας:

Bash
pip install -r requirements.txt
Έλεγχος Υλικού (Hardware): Αν σκοπεύετε να εκπαιδεύσετε μοντέλα βαθιάς μάθησης (Deep Learning), μπορείτε να χρησιμοποιήσετε το παρεχόμενο notebook GPU_Test.ipynb για να επιβεβαιώσετε ότι η κάρτα γραφικών σας αναγνωρίζεται σωστά από τα frameworks (π.χ. PyTorch/TensorFlow).

💡 Οδηγίες για τους Μαθητές
Exploration: Κάθε φάκελος περιέχει αυτόνομο υλικό. Μπορείτε να ξεκινήσετε από οποιαδήποτε θεματική σας ενδιαφέρει.

Notebooks vs Scripts: Προτιμήστε τα αρχεία .ipynb για διαδραστική μάθηση και τα .py για την κατανόηση της τελικής δομής μιας εφαρμογής.

Documentation: Μην παραλείπετε να διαβάζετε τα σχόλια στον κώδικα, καθώς περιέχουν επεξηγήσεις για τη μαθηματική λογική πίσω από τους αλγορίθμους.
