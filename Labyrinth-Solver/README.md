# 🧩 Labyrinth Solver

Αυτό το project είναι ένας αυτόματος επιλυτής λαβυρίνθου που χρησιμοποιεί αλγόριθμους αναζήτησης. Είναι μέρος της εκπαίδευσής μου στις βασικές αρχές της Τεχνητής Νοημοσύνης (AI).

## 🚀 Περιγραφή
Ο κώδικας δέχεται έναν λαβύρινθο σε μορφή πίνακα (grid) και βρίσκει το συντομότερο μονοπάτι από το σημείο **S** (Start) στο σημείο **E** (Exit).

### Σύμβολα Λαβυρίνθου:
- `S` : Αφετηρία
- `E` : Έξοδος
- `1` : Τοίχος / Εμπόδιο
- `0` : Ελεύθερη διαδρομή

## 🧠 Αλγόριθμος: Breadth-First Search (BFS)
Για την επίλυση χρησιμοποίησα τον αλγόριθμο **BFS**. 
- **Πλεονέκτημα:** Εγγυάται τη συντομότερη διαδρομή σε έναν λαβύρινθο χωρίς βάρη.
- **Λειτουργία:** Ο αλγόριθμος εξερευνά όλα τα γειτονικά κελιά σε κάθε βήμα, σχηματίζοντας ένα "κύμα" που απλώνεται μέχρι να βρει τον στόχο.



## 🛠️ Τεχνολογίες
- **Language:** Python 3.x
- **Libraries:** `collections.deque` (για βέλτιστη διαχείριση της ουράς/queue)
