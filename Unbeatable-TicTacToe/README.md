# ❌⭕ Unbeatable Tic-Tac-Toe (Minimax AI)

Μια τρίλιζα όπου ο υπολογιστής δεν χάνει ποτέ. Χρησιμοποιεί τον αλγόριθμο λήψης αποφάσεων **Minimax**.

## 🧠 Πώς λειτουργεί;
Ο αλγόριθμος **Minimax** εξετάζει όλα τα πιθανά μελλοντικά σενάρια του παιχνιδιού:
1. Δίνει βαθμολογία σε κάθε αποτέλεσμα (Νίκη = +10, Ήττα = -10, Ισοπαλία = 0).
2. Επιλέγει την κίνηση που **μεγιστοποιεί** το δικό του κέρδος και **ελαχιστοποιεί** την πιθανότητα νίκης του παίκτη.



## 🛠️ Τεχνικά Χαρακτηριστικά
- **Recursion:** Ο αλγόριθμος καλεί τον εαυτό του για να "δει" βαθιά μέσα στο παιχνίδι.
- **Game Theory:** Εφαρμογή βασικών αρχών θεωρίας παιγνίων.

## 🕹️ Οδηγίες
Τρέξτε το αρχείο `tictactoe_ai.py` για να δείτε την AI σε δράση!
