# 🔢 Handwritten Digit Recognizer (Logic)

Αυτό το project εξερευνά πώς οι υπολογιστές μπορούν να "δουν" και να αναγνωρίσουν χειρόγραφους αριθμούς μετατρέποντάς τους σε δεδομένα.

## 🧠 Η Λογική (Computer Vision Intro)
Κάθε εικόνα μετατρέπεται σε έναν πίνακα από πίξελ (pixels). 
- Το `0` αντιπροσωπεύει το λευκό χρώμα (κενό).
- Το `1` αντιπροσωπεύει το μαύρο χρώμα (μελάνι).



## 🛠️ Τεχνολογίες
- **Python Arrays:** Χρήση λιστών για την αναπαράσταση εικόνας.
- **Pattern Matching:** Βασική λογική σύγκρισης δεδομένων.

## 🌟 Επόμενο Βήμα
Ο στόχος είναι να επεκταθεί αυτό το project χρησιμοποιώντας τη βιβλιοθήκη **Scikit-learn** και το dataset **MNIST** για κανονική αναγνώριση με Νευρωνικά Δίκτυα.
