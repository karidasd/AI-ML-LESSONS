# 🔥 Heatmap Visual Analyzer (Computer Vision)

## 📌 Περιγραφή
Αυτό το project είναι μια εφαρμογή **Computer Vision** που αναλύει την αλληλεπίδραση των χρηστών μέσω θερμικών χαρτών (Heatmaps). Ο αλγόριθμος εντοπίζει αυτόματα τα "Hotspots" (περιοχές υψηλής δραστηριότητας) και υπολογίζει το ποσοστό κάλυψης στην οθόνη.

---

## 🧠 Πώς λειτουργεί;

1. **HSV Conversion:** Μετατρέπουμε την εικόνα σε χώρο HSV (Hue, Saturation, Value) για να απομονώσουμε τις αποχρώσεις του κόκκινου χωρίς να επηρεαζόμαστε από τη φωτεινότητα.
2. **Masking:** Δημιουργούμε μια δυαδική μάσκα όπου τα "καυτά" σημεία γίνονται λευκά και τα υπόλοιπα μαύρα.
3. **Statistical Extraction:** Υπολογίζουμε το **Heatmap Index**, δηλαδή το ποσοστό της σελίδας που τραβάει την προσοχή του χρήστη.
